# Condicionais (if) ou (se)
- if(teste lógico){verdadeiro}
- if(teste lógico){verdadeiro} else {falso}
- if(teste lógico){verdadeiro} else if(teste lógico) {verdadeiro} else {falso}
- else = senão

# Teste lógico
- (==) igual
- (!=) diferente
- (>) maior
- (<) menor
- (>=) maior ou igual
- (<=) menor ou igual

- && => AND => E
- || => OR => OU
- ! => NOT => NÃO
